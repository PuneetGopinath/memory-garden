# Memory Garden

Memory Garden, a digital space that chronologically stores your memories using AI. Memories shouldn't feel like files, they should feel alive.

## Features

- AI-generated memory insights
- Interactive timeline visualization
- Smart AI-generated tags
- Markdown support for memory descriptions
- Secure authentication and image uploads with Supabase
- Organize memories by date
- Private Capsules *(UPCOMING)*

## Tech Stack

- React (with Vite)
- Supabase
    - Authentication
    - PostgreSQL Database
    - Storage
    - Edge Functions
- Tailwind CSS
- Google Gemini API
